# 🎵 Папка для музыки

Сюда кладите свои аудиофайлы для использования в плеере DnD Chronicles.

## Поддерживаемые форматы
- `.mp3` — рекомендуется
- `.ogg`
- `.wav`
- `.flac`
- `.m4a`

## Как использовать
1. Откройте приложение (index.html)
2. Нажмите кнопку 🎵 в навигации
3. В плеере нажмите **«Добавить треки»**
4. Выберите файлы из этой папки (или любой другой)

## Плейлисты
- Нажмите **«🎶 Плейлист»** чтобы создать новый плейлист
- В списке треков нажмите **+** рядом с треком и выберите плейлист
- Переключайтесь между плейлистами через вкладки в плеере

> Треки хранятся в памяти браузера в рамках сессии.
> При перезагрузке страницы нужно добавить их снова.
